# JPQL #
---
`JPQL`은  기존 데이터베이스내의 테이블을 조작하는 `SQL`과는 달리  `JPA`기술로 생성된 자바 엔터티를 조회하는 언어이다.
사실상 엔터티와 테이블레코드는 `JPA`기술로 동기화 되긴하나 , 다르다 .
